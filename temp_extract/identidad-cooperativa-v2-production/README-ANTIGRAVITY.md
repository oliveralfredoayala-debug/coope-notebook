## Integración

1. Verificar que no exista otro artefacto con el Capsule UUID.
2. Copiar la carpeta `kb/` a la raíz del proyecto.
3. Fusionar `content-catalog.json` por UUID.
4. Fusionar `uuid-registry.json` sin reemplazar entradas existentes.
5. Mantener el banco de preguntas como archivo separado; no renderizarlo como tab estudiantil.
6. Ejecutar el validador de catálogo.
7. Abrir el HTML y comprobar navegación, inner tabs, textarea, modo presentación y responsive.
8. Actualizar el Index únicamente después de confirmar que la ruta existe.
