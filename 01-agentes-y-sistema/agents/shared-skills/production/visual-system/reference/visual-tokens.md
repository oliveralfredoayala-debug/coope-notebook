# Tokens visuales aprobados · CoopeNotebook KB

## Principio

Los colores no son decorativos. Tienen semántica fija.

## Tokens

```css
:root{
  --verde:#234a35;
  --verde-sec:#2f7d52;
  --verde-suave:rgba(47,125,82,.06);
  --verde-borde:rgba(47,125,82,.16);

  --amarillo:#f0a226;
  --amarillo-suave:rgba(240,162,38,.08);
  --amarillo-borde:rgba(240,162,38,.20);

  --morado:#7d6a9c;
  --morado-suave:rgba(125,106,156,.07);
  --morado-borde:rgba(125,106,156,.18);
}
```

## Semántica

- verde: estructura y jerarquía;
- amarillo: recordar;
- morado: legal literal;
- gris: soporte y neutralidad.

## Prohibiciones

- números dentro de círculos;
- fondos saturados extensos;
- morado fuera de contenido legal literal;
- amarillo como categoría general;
- verde como bloque decorativo dominante.
