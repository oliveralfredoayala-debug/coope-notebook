# 00 · Núcleo del proyecto

## Qué es esto
**Cooperativa Notebook** — sistema de contenido teórico para la formación de facilitadores cooperativistas hondureños. Las piezas se proyectan en talleres (no se reparten al alumno).

## La pieza atómica: la cápsula KB
Una **cápsula KB** (Knowledge Base) es la unidad mínima del contenido teórico de Course-inator. Es la "presentación de taller" del facilitador: su PowerPoint.

- Se **proyecta**, no se reparte.
- **No** tiene niveles de dificultad.
- **No** captura respuestas del alumno.
- Cada cápsula = **UN tab / UNA lámina**.
- Varias cápsulas de un tema se ensamblan en una presentación con índice vertical a la izquierda.

## Ecosistema
- **Course-inator / Jester** — la plataforma donde el facilitador ordena y proyecta las cápsulas. El orden de las cápsulas lo define el facilitador en Jester.
- **Cooperativa Notebook** — la marca/identidad de origen.

## Principio rector
> Una cápsula = una idea que cabe proyectada en una pantalla sin agobiar.

## Relación con otros skills
Este sistema se aparta a propósito del skill `cooperativa-notebook` (que prohíbe tabs y usa modos Guided/Balanced/Expert). Aquí: **sí hay tabs verticales, no hay niveles.** No mezclar ambos formatos.
