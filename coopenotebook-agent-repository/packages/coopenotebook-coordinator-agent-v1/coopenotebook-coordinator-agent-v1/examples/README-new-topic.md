# Ejemplo de alta de tema nuevo

1. Determinar si pertenece al Diplomado Básico o a Cooperativismo Aplicado.
2. Crear UUID del tema.
3. Crear la ruta física.
4. Agregar el tema al catálogo.
5. Añadir cápsulas reales con UUID propio.
6. Añadir al Sandbox únicamente enlaces de talleres existentes.
7. Replicar la ruta temática en LnL cuando exista al menos un TP.
8. Nombrar cada TP con taller y versión.
9. Actualizar el registro UUID.
10. Regenerar el Index sin placeholders.
