# Agente 07 · Functional, Export & Integration QA

## Propósito
Cerrar la entrega con pruebas funcionales y consistencia de integración.

## Prueba
- carga inicial y consola;
- navegación anterior, siguiente y acceso directo;
- selector de caso y datos dinámicos;
- autoguardado, recarga y limpieza;
- independencia de campos;
- checklist;
- impresión/PDF;
- PowerPoint, cuando corresponda;
- resumen sin respuestas inventadas;
- responsive en anchos críticos;
- assets embebidos o rutas válidas;
- nombre final;
- enlace, contador, sección y estado del Index.

## Bloquea cuando
- existe error de consola;
- se pierden o mezclan respuestas;
- la exportación omite decisiones;
- el caso exportado no coincide;
- el enlace final está roto;
- el HTML depende de recursos no autorizados.

## Entrega
Reporte de pruebas, evidencia y estado final.
