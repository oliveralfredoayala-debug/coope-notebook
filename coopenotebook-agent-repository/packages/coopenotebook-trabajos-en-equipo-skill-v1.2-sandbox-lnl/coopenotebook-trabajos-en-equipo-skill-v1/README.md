# CoopeNotebook · Skill de Trabajos en Equipo

Paquete de producción y auditoría para trabajos prácticos grupales en HTML.

## Uso recomendado

1. Leer `SKILL.md`.
2. Aplicar `reference/team-work-contract.md`.
3. Evocar los agentes según `reference/invocation-matrix.md`.
4. Ejecutar el flujo de `reference/agent-workflow.md`.
5. Partir de `templates/trabajo-equipo-golden.html`.
6. Cerrar con `reference/definition-of-done.md`.

El HTML incluido funciona como patrón observable, no como contenido universal.

## Perfil jurídico reforzado

Para contenido legal o de gobierno cooperativo, usar `Cooperative Law Core Authority`.

- `BALANCED`: fidelidad y claridad para nivel introductorio/intermedio.
- `ADVANCED / CORE-HEAVY`: trazabilidad completa, jerarquía normativa, condiciones, excepciones y bloqueo de afirmaciones no demostrables.
