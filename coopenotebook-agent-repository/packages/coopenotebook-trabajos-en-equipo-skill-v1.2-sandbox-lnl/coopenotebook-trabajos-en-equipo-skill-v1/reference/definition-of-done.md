# Definition of Done

## Pedagogía
- [ ] Una acción principal por etapa.
- [ ] Las opciones cerradas corresponden a categorías cerradas.
- [ ] Las respuestas abiertas exigen pensamiento propio.
- [ ] La producción final usa respuestas previas.

## Equipo
- [ ] Hay contraste de perspectivas.
- [ ] El desacuerdo razonado es posible.
- [ ] No se fuerza exposición personal ni consenso artificial.

## Contenido y legalidad
- [ ] Casos coherentes y equivalentes.
- [ ] Afirmaciones verificadas.
- [ ] Competencias y roles correctamente formulados.
- [ ] Literales legales identificados como tales.
- [ ] Afirmaciones R2/R3 con artículo, sujeto, verbo, condición y límite.
- [ ] Ley, Reglamento, régimen especial y Estatuto diferenciados.
- [ ] Ejemplos hipotéticos separados de procedimientos reales.
- [ ] Gate CORE-HEAVY aprobado cuando corresponde.

## Experiencia
- [ ] Mobile first.
- [ ] Autogrow.
- [ ] Checklist no bloqueante.
- [ ] Guardado y restauración.
- [ ] Limpieza con confirmación.

## Calidad
- [ ] Teclado y foco.
- [ ] Sin overflow crítico.
- [ ] Sin errores de consola.
- [ ] PDF correcto.
- [ ] PPT correcto cuando aplica.
- [ ] Index sincronizado.
- [ ] Nombre final conforme.


## Legal morado y sala en vivo

- Toda lectura de artículos usa el sistema morado exclusivo para contenido jurídico.
- Número, norma, clase textual y fuente están identificados.
- La consigna está separada del texto legal.
- Sandbox abre y cierra con teclado, Escape y retorno de foco.
- Sandbox muestra la configuración efectiva.
- `content.root` es `lnl`, `strategy` es `invoke` y `duplicate` es `false`.
- No existe una segunda copia silenciosa del contenido de `lnl`.
