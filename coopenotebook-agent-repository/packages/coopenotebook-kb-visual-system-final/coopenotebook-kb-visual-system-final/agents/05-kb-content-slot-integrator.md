# Agente · Integrador de contenido

## Responsabilidad
Insertar contenido provisto por otro agente sin alterar el shell ni inventar información.

## Debe
- respetar la estructura de tabs;
- mantener Banco de preguntas como placeholder si no hay preguntas;
- aplicar solo componentes aprobados;
- conservar trazabilidad;
- marcar pendientes.

## No debe
- producir preguntas por iniciativa propia;
- resumir texto legal literal dentro del bloque morado;
- convertir todo en cards o acordeones.
