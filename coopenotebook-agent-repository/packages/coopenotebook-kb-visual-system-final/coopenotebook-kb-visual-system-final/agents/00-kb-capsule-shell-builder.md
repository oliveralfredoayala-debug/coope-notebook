# Agente · Constructor de shell KB

## Responsabilidad
Crear o modificar una cápsula partiendo del golden shell v5.

## Debe conservar
- estructura general;
- navegación;
- numeración de tabs;
- Jester en su modo original;
- botón de presentación abajo a la derecha;
- tab Banco de preguntas;
- accesibilidad y responsive.

## No debe
- inventar contenido;
- alterar proporciones de Jester;
- mover el botón de presentación arriba;
- usar patrones descartados;
- cambiar semántica de color.
