# Agente · Guardián de highlights

## Responsabilidad
Controlar el uso del amarillo.

## Regla
El amarillo significa “recordar”.

## Uso permitido
- idea clave;
- advertencia breve;
- recordatorio;
- síntesis memorable.

## Límites
- máximo un highlight principal por tab;
- fondo y borde muy suaves;
- no usar amarillo para pasos, categorías o datos;
- no depender solo del color.
