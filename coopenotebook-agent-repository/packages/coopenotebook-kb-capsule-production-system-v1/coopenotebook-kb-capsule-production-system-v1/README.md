# CoopeNotebook KB Capsule Production System

Skill completa para producir, auditar y preparar cápsulas KB.

## Incluye

### Agentes
- auditor de contenido, evidencia y pedagogía;
- auditor lingüístico y de caracteres;
- auditor de presentación visual;
- auditor de shell, navegación, accesibilidad e integridad técnica;
- auditor legal verbatim;
- agente de banco de preguntas reutilizable.

### Sistema visual
- golden shell v5;
- biblioteca visual aprobada;
- tokens semánticos;
- reglas de color;
- reglas de densidad;
- referencia visual de cápsula completa;
- selector visual curado.

### Contratos
- shell;
- QA;
- legal;
- visual;
- flujo de agentes.

## Nota
El orquestador es la skill. Los archivos en `agents/` son funciones especializadas.
