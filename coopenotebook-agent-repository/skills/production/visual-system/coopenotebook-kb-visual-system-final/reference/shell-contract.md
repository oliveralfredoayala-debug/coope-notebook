# Contrato del shell v5

## Elementos congelados

- Jester en composición original.
- Bubble a la derecha del personaje.
- Sidebar con:
  - nombre del taller;
  - nombre de la cápsula;
  - tabs numeradas.
- Apertura como tab `00`.
- Banco de preguntas como tab final.
- Modo presentación en esquina inferior derecha.
- Navegación anterior/siguiente.
- Responsive y accesibilidad base.

## Campos editables

- título HTML;
- nombre del taller;
- nombre de la cápsula;
- nombres y cantidad de tabs;
- contenido interno de tabs;
- recursos visuales aprobados;
- fuentes y metadatos.

## Campos no editables sin autorización

- arquitectura del layout;
- posición del botón de presentación;
- proporción de Jester;
- patrón de navegación;
- estilos de foco;
- mínimos táctiles;
- estructura del placeholder del banco de preguntas.
