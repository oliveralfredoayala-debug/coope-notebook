# CoopNotebook Agent Repository

Monorepo de agentes, skills, contratos, plantillas y activos de CoopNotebook.

Empieza por [`START-HERE.md`](START-HERE.md).

## Modelo operativo

- ChatGPT/OpenAI: coordinación, arquitectura y producción.
- Claude: lectura profunda, auditoría y revisión.
- Antigravity: operación local, pruebas, commit y push a ramas.
- Humano: aprobación y merge.

Los paquetes originales están preservados en `packages/`.
