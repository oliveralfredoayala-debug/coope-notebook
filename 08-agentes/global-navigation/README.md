# Global Navigation Agent

Autoridad única del shell global de CoopeNotebook.

## Ruta canónica

`/08-agentes/global-navigation/`

## Controla

- top bar global;
- identidad Jester + CoopeNotebook;
- nombre corto del área;
- Explorar y Búsqueda global;
- tabs del contenido a la izquierda;
- drawer de artefactos a la derecha;
- carga del artefacto en el área principal;
- footer mínimo;
- responsive, accesibilidad y estados de navegación.

Los demás agentes deben entregar contenido sin menús globales conforme al contrato de `reference/menu-free-content-handoff-contract.md`.
