# Contrato Sandbox ↔ Sala en vivo

## En KB

El tab `Sandbox` solo renderiza enlaces disponibles para el tema.

Cada elemento mínimo contiene:

- UUID del enlace;
- UUID de la KB;
- UUID del taller;
- texto visible;
- URL/ruta real;
- estado `published`.

## En el destino Sandbox

La ficha enlazada puede explicar:

- objetivo del taller;
- dinámica esperada;
- materiales o roles;
- qué verá el grupo en Sala en vivo;
- preparación del facilitador;
- vínculo de lanzamiento cuando corresponda.

## Validaciones

- Ningún enlace huérfano.
- Ningún taller inexistente.
- Ninguna entrada futura o “coming soon”.
- La KB no duplica la configuración completa.
- La relación debe poder recorrerse en ambos sentidos.
