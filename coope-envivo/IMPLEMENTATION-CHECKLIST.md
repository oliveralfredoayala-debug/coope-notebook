# Checklist de implementación

## Firebase

- [ ] App web creada.
- [ ] Email/Password activado.
- [ ] Anonymous activado.
- [ ] Firestore creado.
- [ ] Reglas desplegadas.
- [ ] Índices desplegados.
- [ ] Dominios autorizados revisados.
- [ ] App Check planificado.

## Vercel

- [ ] Repositorio importado.
- [ ] Variables de entorno agregadas.
- [ ] `NEXT_PUBLIC_APP_URL` configurada.
- [ ] Dominio final probado.
- [ ] Preview deployments protegidos si contienen datos reales.

## Prueba en vivo

- [ ] Facilitador crea sala.
- [ ] Participante 1 entra.
- [ ] Participante 2 entra.
- [ ] Actividad aparece sin recargar.
- [ ] Respuestas llegan.
- [ ] Resultados se ocultan.
- [ ] Resultados se muestran.
- [ ] Proyector funciona.
- [ ] Celular pequeño funciona.
- [ ] Reconexión funciona.
