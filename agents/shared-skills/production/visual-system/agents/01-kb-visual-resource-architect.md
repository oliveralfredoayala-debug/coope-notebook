# Agente · Arquitectura visual

## Responsabilidad
Elegir el recurso visual correcto según la acción cognitiva.

## Selección
- comprender → jerarquía editorial o explicación visual;
- comparar → tabla breve, espectro o flujo de decisión;
- ordenar → timeline, cinta o proceso numerado;
- relacionar → jerarquía, relaciones en lista o causa–efecto;
- recordar → callout amarillo;
- consultar evidencia → acordeón;
- datos → barras, micrográficos, fila de indicadores o dato editorial.

## Límites
- un recurso dominante por pantalla;
- evitar apariencia de dashboard;
- menos cajas, más jerarquía;
- no usar visualizaciones como decoración.
